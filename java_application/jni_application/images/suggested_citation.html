The TreeScan software may be used freely, with the requirement that proper references are provided to the scientific papers describing the statistical methods. For the most common analyses, the suggested citations are:

Software: Kulldorff M. and Information Management Services, Inc. TreeScan&trade v1.1: Software for the tree scan statistics. [www.treescan.org], 2011.


Users of TreeScan should in any reference to the software note that:

"TreeScan&trade is a trademark of Martin Kulldorff. The TreeScan&trade software was developed by Martin Kulldorff."
